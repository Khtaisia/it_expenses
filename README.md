# IT Stack Analyzer

Веб-сервис для анализа технологических стеков IT-проектов. Помогает выявлять популярные комбинации инструментов и технологий в разработке и визуализировать их в виде графиков.

Ссылка на рабочий проект: [https://ваш-логин.pythonanywhere.com](https://ваш-логин.pythonanywhere.com)

## Технологии
- Python 3.x
- Django 6.0
- Pandas (анализ данных)
- Plotly / Matplotlib (визуализация)
- Bootstrap 5 (интерфейс)
