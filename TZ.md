# Техническое задание на проект: IT Stack Analyzer

## 1. Цель проекта
Создание веб-сервиса для анализа технологических стеков IT-проектов. Сервис помогает выявлять популярные комбинации инструментов и технологий, используемые в разработке, и визуализировать их в виде графиков.

## 2. Роли пользователей
- Гость: просмотр аналитики и популярных комбинаций технологий.
- Зарегистрированный пользователь: добавление новых проектов с описанием используемых технологий, просмотр собственной статистики.
- Администратор: управление всеми проектами и технологиями через админку Django.

## 3. Модели данных
1. Technology: описание технологии (Python, Django, React и т.д.)  
   - Поля: name, category (например: Backend, Frontend, DB)
2. Project: карточка проекта  
   - Поля: name, description, created_by (ForeignKey к User), date_added
3. ProjectTechnology: связь многие-ко-многим между проектами и технологиями  
   - Поля: project (ForeignKey), technology (ForeignKey)

## 4. Ключевой функционал
- Добавление нового проекта с выбором технологий.
- Просмотр списка проектов и фильтрация по технологиям.
- Анализ и визуализация популярных технологий и комбинаций (гистограммы, диаграммы).
- Экспорт аналитики в виде графиков.

## 5. Внешние интеграции / Аналитика
- Библиотеки для анализа данных: Pandas (сбор статистики, подсчёт комбинаций)
- Визуализация: Plotly или Matplotlib
